    A.Pagination Class

    Extends: A.Component
    Defined in: src/aui-pagination/js/aui-pagination.js:36
    Module: aui-pagination

    A base class for Pagination, providing:

    • Widget Lifecycle (initializer, renderUI, bindUI, syncUI, destructor)
    • Set of controls to navigate through paged data

    Check the live demo.

    Defined in src/aui-pagination/js/aui-pagination.js:71
